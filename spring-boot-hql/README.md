# Spring Boot HQL Demo

Project demonstrates Hibernate Query Language (HQL) operations on a `Restaurant` entity.

Database: `fsadendexam` (H2 file DB at `~/fsadendexam`)

Run:

```bash
cd spring-boot-hql
mvn spring-boot:run
```

The `ClientDemo` class inserts two `Restaurant` records and then updates one record using HQL with named parameters.
